(부모)
.flex-container

  • flex-direction :

    배열 형태를 설정

  • flex-wrap :

    Flex Items의 여러 줄 묶음(줄 바꿈) 설정

  • flex-flow :

    flex-direction과 flex-wrap의 단축 속성

  • align-content :

    flex-direction : Column 정렬 방법을 설정(2줄 이상)

  • justify-content :

    flex-direction : Row 정렬 방법을 설정

  • place-content :

    justify-content과 align-content의 단축 속성

  • align-items :

    교차 축(cross-axis)에서 Items의 정렬 방법을 설정(1줄)

(자식)
.in_box

  • flex-grow :

    여백 공간이 남을때 공간만큼 확장하기

  • flex-shrink :

    여백의 공간이 없을때 축소하기

  • flex-basis :

    row 일때 Width / column 일때 Height 초기 값 설정.

  • flex :

    flex-basis와 flex-grow와 flex-shrink의 단축 속성

  • order :

    위치 재배열 명령어
