Scope : container

  • flex-direction :

    Flex Items의 주 축(main-axis)을 설정

  • flex-wrap :

    Flex Items의 여러 줄 묶음(줄 바꿈) 설정

  • flex-flow :

    flex-direction과 flex-wrap의 단축 속성

  • justify-content :

    주 축(main-axis)의 정렬 방법을 설정

  • align-content :

    교차 축(cross-axis)의 정렬 방법을 설정(2줄 이상)

  • align-items :

    교차 축(cross-axis)에서 Items의 정렬 방법을 설정(1줄)

Scope : items

  • flex-basis :

    가로 값을 정해주는 역활을 한다.

  • flex-shrink :

    가로 값을 정해주는 역활을 한다.
