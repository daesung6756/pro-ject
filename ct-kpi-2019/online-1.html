문자열 자료형

자바스크립트 프로그래밍 언어는 => 안녕하세요.
위와 같은 문장을 문자열로 인식하지 못한다.
자바스크립트 언어에서 문자열 자료형으로 입력하기 위해서는

ex)
'안녕하세요.' (작은 따옴표)
"안녕하세요." (큰 따옴표)

둘중에 어느것을 사용해도 상관 없다.
소속되어있는 팀 코딩 컨벤셜에 맞춰서 사용하면 된다.
단, 예외 적으로

"'안녕', 이라고 말했습니다."
=> 문장 자체에 작은 따옴표가 들어가는 경우 문자열로 바꿀때에는 큰따옴표를 사용하면 된다.
'"안녕", 이라고 말했습니다.'
=> 반대일 경우에는 반대로 사용하면 된다.

문자열 내부에 역슬래시를 사용을 하게 되면 역슬래시와 역슬래시 다음에 오는 문자를 하나로 인지하여
특수 처리를 하게 되는데 이를 '이스케이프 문자'라고 한다.
이스케이프 문자를 활용하면 큰따옴표 한가지로 문자열을 완성 할 수 있다.
대표적으로 기억해야될 이스케이프 문자는 다음과 같다.

"\t" => 탭
"\n" => 줄바꿈
"\'" => 작은따옴표
"\"" => 큰따옴표
"\\" => 역슬래시

* 글자 서체를 "Arial" 지정하면 원화 표시가 아닌 역슬래시 표현이 된다.
* 일부 웹브라우저에서는 "\t"이 출력 되지 않습니다.


문자열 연결 연산 천리

"안녕" + "하세요"
=> 문자열끼리 덧셈 기호를 활용하여 연결을 할 수가 있다.

출력시
=> "안녕하세요" 라고 출력 된다.
