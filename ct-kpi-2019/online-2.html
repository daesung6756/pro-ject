변수

변수는 프로그래밍 언어 전체에서 가장 중요한 개념이다.
코드가 진행 될때 값을 저장할때 사용하는 식별자이다.
var(variable)을 통해 변수를 선언하는 방법은 아래와 같다.

- 값을 할당하지 않고 선언을 하는 경우
var 식별자;
=> 위와 같이 값을 할당하지 않으면 undefined 를 출력한다.

- 값을 할당하고 선언하는 경우
var 식별자 = 10;
=> 위와 같이 값을 할당 하면 10 을 출력한다.

* html, css, script등 모든 언어는 처음부터 차례대로 실행된다.
* 스크립트에서 처음 선언하는 할당값을 초기화라고 한다.

- 값을 할당하고 선언하는 경우
var 식별자 = 10 (초기화 값);

- 값을 할당하지 않고 선언을 하는 경우
var 식별자;
식별자 = 10 (초기화 값)


변수와 함께 사용하는 연산자 (산술, 문자열 연결, 증감, 대입 등)

1. 산술연산자
표준 덧셈, 뺄셈, 곱셈, 나누기 등이 있으며, 숫자 자료형에 사용 할 수 있다.

ex) 덧셈
var 변수 = 10; // 변수를 선언하고 10이라는 값을 할당
변수 + 10; // 변수에 + 10을 더한다.
=> console.log(변수) 20을 출력 한다.


2. 문자열연산자
문자열 연결 연산자라고 하며, 덧셈을 사용하여 문자열과 문자열은 연결 시키는 역할을 한다.

ex) 문자열 연결 연산
var 변수 = "안녕하세요."; // 변수를 선언하고 "안녕하세요." 라는 값을 할당
변수 + "반갑습니다."; // 변수에 + 10을 더한다.
=> console.log(변수) "안녕하세요.반갑습니다."을 출력 한다.


3. 증감연산자
대상을 1씩 증가시키거나, 1씩 감속시키는 역할을 한다.

ex)
var 변수 = 10; // 변수를 선언하고 10 이라는 값을 할당
변수++; // 변수 후위에 + 1을 더한다.
++변수; // 변수 전위에 + 1을 더한다.
변수--; // 변수 후위에 - 1을 뺀다.
--변수; // 변수 전위에 - 1을 뺀다.


4. 대입연산자
대입 연산자는 변수에 값을 대입할 때 사용하는 이항 연산자이다.
= // 왼쪽에 오른쪽 피연산자의 값을 대입함.
+= // 왼쪽에 오른쪽 피연산자의 값을 더하고 왼쪽에 대입.
-= // 왼쪽에 오른쪽 피연산자의 값을 빼고 왼쪽에 대입.
%= // 왼쪽에 오른쪽 피연산자의 값을 나누고 왼쪽에 대입.
*= // 왼쪽에 오른쪽 피연산자의 값을 곱하고 왼쪽에 대입.
대입 연산자에서 연산자의 순서는 매우 중요하므로 주의를 기울여야 한다.
